# Sunsetsurf
## Pasos para levantar la app
```
1 --> Iniciar La Maquina virtual
2 --> Iniciar Xampp y arrancar los servicios de MySQL y de Apache Server
3 --> ir al enlace del escritorio que te lleva a SunsetSurf, nuestra app
```
